Hinweise zum Datenschutz

Bitte beachten Sie die Datenschutzerklärung des Landesrechnungshofs Schleswig-Holstein.

Der Schutz Ihrer persönlichen Daten ist uns wichtig.
Nachfolgend möchten wir Sie daher über die Verarbeitung Ihrer Daten bei der Nutzung der App informieren.

Diese App wird auf GitHub Pages gehostet. GitHub (GitHub, Inc., 88 Colin P Kelly Jr St San Francisco, CA 94107 United States) erfasst und speichert die IP-Adressen der Besucher dieser Website aus Sicherheitsgründen (link). Die Server von GitHub befinden sich in den USA. Deshalb werden Ihre Daten, die für den Aufruf der Webseite benötigt werden (Zugriffsdaten), in die USA übermittelt. Github stellt aber sicher, dass dies in Übereinstimmung mit der DSGVO geschieht (vgl. Art. 6 (1) f. DSGVO). Um den Schutz Ihrer Daten sicherzustellen, bedienen wir uns der Standardvertragsklauseln der EU-Kommission gemäß Art. 46 Abs. 2 lit. c) DSGVO. Diese sind unter https://github.com/customer-terms/github-data-protection-agreement abrufbar. Hinweise zum Datenschutz bei GitHub und zur Konformität mit dem Datenschutzrecht finden Sie hier.

Darüber hinaus erheben wir keine Benutzerdaten. Die App verwendet keine Cookies oder Trackingtechnologien. Auf die von GitHub gespeicherten IP-Adressen haben wir keinen Zugriff. Sämtliche Berechnungen der WebApp finden lokal im Browser statt; eine Kommunikation mit einem Server ist hierfür nicht erforderlich. Das bedeutet insbesondere, dass wir nicht sehen können, welche verschiedenen Szenarien die Benutzer auf dieser App durchspielen.

Diese Datenschutzerklärung kann gelegentlich aktualisiert werden, um Änderungen in der App oder gesetzlichen Anforderungen zu berücksichtigen. Die aktuelle Version der Datenschutzerklärung ist stets auf unserer Webseite verfügbar.
